OISY — send flow: personal note (wireframe)

Send

Layout/interaction wireframe of the send flow. Empty state shows a "+ Add" button (like the details modal); once a note exists, the row shows the text with an edit pencil. Length validation (demo limit 50; real 500, error-only); Clear empties without leaving edit mode; OK confirms, Cancel discards. Toggle "Simulate: at note cap" to see the disabled add + message. The note shows read-only on review. On Mobile the entry opens as a bottom sheet.
